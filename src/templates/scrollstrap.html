About us

Welcome to the NYC Python Group, dedicated to promoting Python in the NYC community. This group is for anyone interested in Python software development, where all levels of knowledge are welcome to collaborate, learn, and share ideas. We hold a number of events each month, including tutorial events, nights-of-talk, lightning-talks, micro-hackathons, study groups, API nights, hack-and-tells, weekly office hours, and project nights. We typically meet on Tuesdays, Saturdays, and Sundays.
